🤔 Flexboxとは何か？

Flexible Box Layout（柔軟なボックスレイアウト）

Flexboxは、要素を一列に並べたり、空間を効率的に配分したりするためのCSS レイアウト手法です。

❌ 従来の方法

• float を使用
• position で調整
• 複雑で難しい
• レスポンシブが困難

✅ Flexbox

• display: flex; だけ
• 自動的に配置
• シンプルで直感的
• レスポンシブ対応

📝 重要なポイント

親要素（Flexコンテナ）：display: flex; を指定

子要素（Flexアイテム）：親要素の直接の子要素が自動的にFlexアイテムになる

🏠 親 → 📦📦📦 子たち

🏗️ 基本構造とコード

HTML構造

<div class="parent"> <div class="child">子要素1</div> <div class="child">子要素2</div> <div class="child">子要素3</div> </div>

CSS設定

.parent { display: flex; background: #e2e8f0; padding: 20px; } .child { background: #fed6e3; padding: 15px; margin: 5px; }

🔑 キーポイント： 親要素に display: flex; を指定するだけで、子要素が自動的に横並びになります！

⚙️ 主要なプロパティ

justify-content

横方向の配置

justify-content: • flex-start (左寄せ) • center (中央) • flex-end (右寄せ) • space-between (両端揃え) • space-around (均等配置)

align-items

縦方向の配置

align-items: • flex-start (上寄せ) • center (中央) • flex-end (下寄せ) • stretch (伸ばす) • baseline (ベースライン)

flex-direction

並ぶ方向

flex-direction: row (横並び - デフォルト) flex-direction: column (縦並び) flex-direction: row-reverse (横並び逆順) flex-direction: column-reverse (縦並び逆順)
